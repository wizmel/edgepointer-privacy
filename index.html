日本語

EdgePointer（以下「本アプリ」）は、利用者のプライバシーを最優先します。

収集する情報

本アプリは、個人情報を含むいかなるデータも収集・保存・送信しません。

  • ネットワーク通信を行いません（インターネット権限を要求しません）。
  • 解析・広告・トラッキング用の SDK を一切含みません。
  • 第三者とデータを共有しません。

アクセシビリティ機能の利用について

本アプリは、片手では届かない画面領域を操作できるようにする補助ツールです。この目的のため Android のユーザー補助（AccessibilityService）API を使用します。

  • ユーザー補助は、利用者の操作（タップ・スワイプ・長押し）を画面上の任意のアプリへ送る目的にのみ使用します。
  • 画面の内容（テキスト・パスワード・表示要素など）を読み取りません（canRetrieveWindowContent は false）。
  • 取得・記録・外部送信される情報はありません。

権限

本アプリはランタイム権限を要求しません。動作にはユーザー補助の有効化のみが必要です。

English

EdgePointer ("the app") puts user privacy first.

Data we collect

The app does not collect, store, or transmit any data, including personal information.

  • No network access (the app does not request the Internet permission).
  • No analytics, advertising, or tracking SDKs of any kind.
  • No data is shared with third parties.

Use of the Accessibility Service

The app is an assistive tool that lets you operate parts of the screen you cannot reach one-handed. For this purpose it uses Android's AccessibilityService API.

  • The Accessibility Service is used solely to deliver the gestures you make (tap, swipe, long-press) to whatever app is on screen.
  • It does not read screen content (text, passwords, on-screen elements); canRetrieveWindowContent is false.
  • No information is captured, logged, or sent anywhere.

Permissions

The app requests no runtime permissions. It only requires the Accessibility Service to be enabled.
